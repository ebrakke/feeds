# Mobile UX Redesign - Feed Management

## Problem Statement

The current mobile UI has several pain points:
- Import button in header wastes prime real estate (one-time action)
- "Paste YouTube URL" section takes too much space on home screen
- Inbox feature goes unused
- Creating topic feeds from existing channels is cumbersome
- Deep video rabbit holes make it hard to navigate back to origin feed

## Core User Workflow

1. Import watch history â†’ get frequency-based feeds (Heavy Rotation, Regulars, etc.)
2. Browse frequency feeds to discover what's new
3. Create topic feeds by cherry-picking channels from existing feeds
4. Channels live in multiple feeds (frequency + topic)
5. Dip into topic feeds when in a specific mood
